id: standarderc20-gateway
title: 'StandardERC20 gateway'
---

[Token Gateway](/docs/glossary#token-gateway) via which any underlying chain's **ERC-20** token can permissionlessly bridge; the `StandardERC20` gateway contracts deploy a [Standard Arb-Token](/docs/glossary#standard-arbtoken) on the [Child chain](/docs/glossary#child-chain) for each bridged token.
