---
id: l2-to-l1-message
title: 'L2 to L1 Message'
---

A message initiated from within an Arbitrum chain to be eventually executed on [Layer 1 (parent chain)](/docs/glossary#layer-1-l1) (e.g., token or Ether withdrawals). On Rollup chains like [Arbitrum One](/docs/glossary#arbitrum-one), the [Challenge Period](/docs/glossary#challenge-period) must pass before a child-to-parent chain message is executed.
